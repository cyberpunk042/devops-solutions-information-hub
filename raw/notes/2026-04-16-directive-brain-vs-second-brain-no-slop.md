## The Distinction (operator-defined)

- **Brain** = per-project agent configuration files. CLAUDE.md, AGENTS.md, DESIGN.md, SOUL.md, skills, hooks, commands, settings. Every project has its own brain. The brain is what constitutes the agent.
- **Second brain** = the shared research wiki. The knowledge system all projects consume from and contribute to. ONE second brain exists; each project has its own brain.

## The Rule Going Forward

1. NEVER call the second brain "the brain." It is ALWAYS "the second brain" or "the research wiki."
2. When addressing a sister project: acknowledge they have THEIR OWN brain. The second brain is a separate system they consume from.
3. Tool output must contain SMART CONTENT — every line either changes behavior, prevents an error, or points to an action. No marketing slop ("central intelligence hub"), no vague infrastructure claims.
4. Think about what you write back to clients. The orient output is the FIRST THING an agent reads. It sets the tone.
